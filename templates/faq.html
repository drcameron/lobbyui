The Basics

Introduction

UnTap is a multiplayer CCG deck tester. Decks are built and play-tested from various formats against other opponents online. Our unique deck builder uses API searches to build your deck ready to play-test simple and mess free. Everything is in the browser no downloads or plugins required.

Deck Building

Deck builder is accessed through creating a deck from the Decks menu and creating or editing the selected deck.

Each CCG type has its own set deck field options, but for the most part all cards are added in the same way.

Basic Input

DECK INPUT
4 Mystic Enchant
4 Flaming Sword
2 Dragon Breath
EXPLAINED

Each Line is prefixed with a quanity followed by the name of the card. This applies to all card input areas (Main/Secondary deck, Sideboard, Alt Cards)

Some cards will require special inputs

Importing a Deck

There are two methods for importing decks uploading a deck file or fetching from a deck URL. Uploading Deck files supports txt dec l5d o8d file types.
Fetching decks from URL works by pulling a deck list down from a URL, if the page has a deck list on it, UnTap will do its best to import it. Not all URLs are supported, please make sure that the deck page is public and login is required to access the page.


Custom Cards

Since UnTap uses remote URLs for all its card images you can by pass the card search API and manually add a custom image URL for a card. Custom cards will flag the deck as containing custom cards to prevent people changing card text, or atleast making you cautious to the know that the card is custom. A three character hash is created to store the URL for future saves.

DECK INPUT
4 Token Frog^http://url.img/tokenfrog.jpg
4 Red Monkey^http://custom.art/monkey.jpg
2 Rules card^http://rules.com/card.jpg
EXPLAINED

Each line is still prefixed with a quantity, but appending ^ then a URL to the card name will bypass the API search

Double Sided Cards

Certain cards will need to be double sided, in this case you can use the % seperator in the deckbuilder. In game once a card is in play it can be fliped to reveal its second face.

DECK INPUT
4 Catterpiller%Butterfly
4 Heads%Tails
EXPLAINED

Each line is still prefixed with a quantity, but splitting the 2 faces with a % will make both side of the card accessible when play-testing.

Game UI

Interface

Flick through the slide show belwow for a basic run down.

  • Play Area: Cards can be dragged around, click for menu, double click to pivot card.
  • Deck: Click for menu and actions, double click for single draw.
  • Hand: Not visible to opponent. Click for card actions, single click to play.
  • Discard, expelled and facedown piles: Click black bar to expand or mousewheel to scroll through cards.
  • Game Log: Shows all actions taken and includes the game chat. P for Public chat, S for spectator chat.

All of your cards will have a left click menu. This menu allows actions to be performed on that card. Actions that send a card back to deck will send the card to its default deck.

Once cards are in the play area they can be moved around via click and drag. To move a pile of cards hold shift while clicking and dragging.

Some games require multiple decks. The deck on the left will always be the default draw deck. The secondary deck will be on the right. Both decks are subject to similar actions but hotkeys will not work for the second deck.

Cards in the play area can be targeted while holding shift and clicking the source card then the target cards.

Most objects in the game when hovered will have a tool tip in the bottom left hand corner. This aids game play and eases the learning curve of the UnTap.in play-test engine.

The game log is for displaying history of events during the game. All cards in the game log can be hovered to show full card. Card names will display as Unknown Card if you are not permitted to see the card in its current location, but will then update once you have permission to see the card.

Hot Keys

Designed to speed up game play, Hot Key offer simple actions to save time opening menus. There are two kinds of hot keys on the UnTap UI.

Global Hot Keys: Can be used at any time, typicaly trigger game events (changeing counters, opening menus and more).

Hover Card Hot Keys: These hot keys can be pressed while hovering cards, they will perform an action based on the hovered card.

Global Hot Keys
  • ESC - Open Game menu
  • Up - Increase HP Counter
  • Down - Decrease HP Counter
  • Shift+Up - Increase HP Counter by 5
  • Shift+Down - Decrease HP Counter by 5
  • N - Next phase of turn
  • X - Unpivot all cards
  • C - Draw from deck
  • F - Find card in deck
  • G - Look at top card of deck
  • V - Shuffle deck
  • ~ - Game Connection Stats
Hover Card Hot Keys
  • Space - Default Action for hovered card
  • T - Send to top of deck
  • Y - Send to bottom of deck
  • J - Clone token card
  • D - Discard card
  • S - Expel card
  • R - Return to hand/Send to end of hand
  • P - Send to face down pile
  • K - Flip Card
  • L - Clone Card
  • Z - Pivot group

Some of these actions will also work while hovering your deck, to actiont the top card of deck.

Voice Chat

Voice chat can be started via the game menu (press ESC or click your username). The feature requires your browser to support WebRTC. Latest Chrome and Firefox support this. To see if your browser supports this Click Here

Other Stuff

Connection Issues

Any issues with not being able to start games, unable to select deck, games returning to lobby once loaded or any game drop outs you can diagnose these issues with the following links

Test Web Sockets - UnTap requires port 443 for the game cleint. If this is being blocked you will not be able to play-test on UnTap.

Injected Code - Some browser extenstions will inject code that may conflict with the clients ability to run properly, this test will scan for any injected code running on your web browser.

If you are still having issues after you have confirmed both of the above checks are fine, please try another browser before reporting the problem.

Device Support

Currently UnTap.in offers full support for the latest versions of Chrome, Firefox, Safari, Internet Explorer and Opera running on Mac, Linux or Windows.

Tablet Support is in the public stages of testing and offers near next to full support for tablet sized iOS 7 Devices and Android Devices. Running UnTap on a tablet requires it to be installed as a Web App.

Installing UnTap.in Web App on iOS 7

Browse to UnTap.in in Safari and select the up arrow box.

Select Add to Home Screen.

Confirm the icon and title loaded correctly and select Add.

UnTap.in will now be on your home screen as a Wep App.


Installing UnTap.in Web App on Android

UnTap.in Support Tablets running Android 4.0+ and Chrome Beta. Follow the bellow instructions to install the web app on your device.

Download Chrome Beta from the Play Store, its free and is required for UnTap to run on Android devices.

Browse to UnTap.in in Chrome Beta and select the three dots.

Select Add to homescreen.

After confirming the app title, select add.

You should now have the app installed on your home screen.
