UnTap Funding

Although UnTap tries to provide as much to the users as possible for free, obviously we have running costs. All funding goes towards paying server costs, and other general upkeep.

By funding you will be open to receiving some "Perks", the main one being access to Server 2. Once you are a funder you will be automatically moved to server 2, a faster server with the dedicated purpose of hosting games. Server 1 & 2 are cross compatable with each other.

Some other perks include up to 6 player multipler games, custom uploaded deck backs, password lock your games, extended game history and of course more deck slots!

Funding is a one time action, although you can fund multiple times to stack deck slots.
